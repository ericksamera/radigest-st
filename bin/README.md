# radigest binaries

Copy deployable radigest binaries here.

Expected first-release binary:

```text
bin/radigest-design
```

Optional later binaries:

```text
bin/radigest
bin/radigest-screen-pairs-cached
bin/radigest-rank-pairs
bin/radigest-plan-depth
```

For Streamlit Community Cloud, `bin/radigest-design` must be a Linux executable for the platform Streamlit runs.
